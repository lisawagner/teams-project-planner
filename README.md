# EVENT CENTRAL - Project Management Tool

Event Central, a project management tool prototype for teams.

## Tech Stack

  1.  React - Frontend
  2.  Redux - State Management
  3.  Thunk - Middleware for asynchronous functions
  4.  Firebase - Backend with:
        - Firestore
        - Cloud functions
        - Authenticaton
        - Deplyment
